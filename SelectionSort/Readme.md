# Selection Sort

Selection Sort is a simple **comparison-based sorting algorithm** used to arrange elements of an array in **ascending order**.

Selection Sort works by **repeatedly selecting the smallest element** from the unsorted part of the array and placing it at the correct position in the sorted part.

The array is divided into two parts:

* **Sorted part** (left side)
* **Unsorted part** (right side)

1. Assume the first element is the minimum

5. Move to the next position
6. Repeat until the array is sorted
