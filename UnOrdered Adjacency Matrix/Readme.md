## 📌 Graph Implementation using Adjacency Matrix (C++)

### 🔹 Overview

This program implements a **graph data structure** using an **adjacency matrix** in C++. It allows basic graph operations such as adding/removing edges, calculating vertex degree, and checking connectivity using Depth First Search (DFS).

---

### 🔹 Features

* Create a graph with user-defined number of vertices
* Add edges between vertices (undirected graph)
* Remove edges
* Calculate the **degree of a vertex**
* Perform **DFS traversal** to check if a path exists between two vertices

---

### 🔹 Where it is used

* Network routing and connectivity problems
* Social network analysis (connections between users)
* Path finding in maps and games
* Graph-based problems in Data Structures & Algorithms
* Computer science concepts like AI search, compilers, and operating systems

---

### 🔹 Why it is used

* Helps represent relationships between objects (nodes & edges)
* Efficient for **dense graphs**
* Simple and easy-to-understand structure
* Useful for implementing traversal algorithms like DFS

---

### 🔹 Requirements

* C++ compiler
* Custom `Stack.h` file (for DFS implementation)

---

### 🔹 How to Run

1. Compile the program
2. Enter number of vertices
3. Enter edges
4. Perform operations like degree calculation or DFS

---

### 🔹 Type of Graph

* Undirected Graph
* Represented using Adjacency Matrix
